# Initial news-source scrape

News websites have a historic record of their news stories generally accesible through indexed pages. The files contained in this folder will help scrappe that historic data for all the registered news sources. 

## Adding a news source (website)

In order to add a new website to the files to be scrapped you'll have to do the following steps.

1. Explore the target website and find the following information.
	- Website url
	- Relevant ''category" url
	- Individual article HTML tags (e.g. h3.td-module-title)
2. create a .json file that contains the information about the new website
3. re build docker file to save changes
extensions**, providing you with some nice features.
